# 能 源 管 理 系 统

# 一、项目概述

# 1.1 项目背景

随着国家“双碳”目标的提出，各地区在不断发展绿色能源的

同时也在探索节能减排的方式方法。而企业作为能源消耗的重点单位，在推进节能减排过程中将是必不可少的一个环节。同时随着企业的发展，能源消耗日益增加，如何实现能源的精细化管理，精准控制能源成本，从而提高产品市场竞争力，已经成为企业自身发展的内在需求。

# 1.2 建设理念

因此借助信息化软件及大数据、云计算、物联网等技术，建设能源管理系统，实现企业能源的可视化管理、精细化管理，帮助企业探索节能降耗潜力和方式，提高能源利用率，降低企业生产成本。

# 1.3 建设目标

1.3.1 能源实时监控，实现自动抄表

1.3.2 能源报表自动生成，数据自动统计

1.3.3 构建企业三级能源计量网络，实现能源可视化管理

1.3.4 通过能源用量趋势分析、对标分析、单耗分析等方法，实现能源的精细化管理，从而达到节能降耗的目的

# 二、系统架构

![image](https://cdn-mineru.openxlab.org.cn/result/2026-04-13/99fff4eb-6a25-44d8-9563-8eef591fe835/54c8ed17403d54df20e4c0f031cb2f674b77bdb3a62b680009d279582f07255a.jpg)


能源管理系统需要采集企业电表、水表、蒸汽表数据，本方案基于 SupOS 搭建能源管理平台，可充分发挥平台的数据采集功能。通过电表、水表、蒸汽表 485 口，将数据直接接入 SupOS 采集中心，平台具备数据解析、数据清洗、数据转换等功能，最终把获取到的数据存入数据库中，实现企业能源的集中监控。

基于 SupOS 平台数据，在能源管理 APP 中实现各类功能，包括利用能源分析模型，构建企业能源三级计量网络，实现能源实时数据的展示、能源动态展示、能源优化调度、能源报表、能耗分析、能源计划监控等功能，帮助管理者掌握企业用能情况，同时挖掘企业节能潜力。

通过 SupOS 标准化接口，可将能源数据共享给 MES 系统、ERP 系统等信息化系统。

# 三、系统功能

作为企业自动化和信息化的重要组成部分，能源管理系统不仅对能源的统一调度、优化资源平衡、提高环保质量、降低产消能耗、掌握生产成本和提高劳动生产率有重要作用，而且对于能源供

需的合理调整和平衡以及在客观信息基础上的能源实际分析、能源计划编制、能源质量管理、能源系统的预测等都是十分有效的。

能源管理系统主要接入工厂水、电、蒸汽表数据，机械表无法实现数据采集功能，需要企业将表更换为智能表。为提高企业的能源管理水平，建议增加大功率设备单独电表，以实现能源的精细化管理。

# 3.1 能源三级管控

构建企业三级能源计量网络，实现能源可视化、精细化管理。一级能源为企业整体能源用量，包括水、电、蒸汽。二级能源为企业各车间、大楼、罐区等，可由企业自行定义。三级能源可为车间的工段、设备、楼层等，由企业自行定义。企业可以组态的方式自定义能源架构图，系统可对各级能源进行趋势查看，掌握能源走势状况。

# 3.3 能源计划管理

# 1) 能源计划

建立能源网络模型或能源控制模型，保证能源供需平衡，编制能源供需计划。根据生产经营计划作出能源消耗计划和外购计划。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-04-13/99fff4eb-6a25-44d8-9563-8eef591fe835/f615ac518958e4df36f3f3325f671d9cfcb2aef6ff226da53cbaded74e412255.jpg)


# 2) 报警管理

系统通过节点状态变化，参考预设参考值，对模拟量的输出、计算点、平均值、变化速率等进行扫描比较，判断状态变化是否异常。将报警信号按照故障事故的重要性进行级别分类，现场以声光报警的予以提醒，同时将向相关人员推送消息、发送手机短信或者语音电话报警，告知报警时间、地点、设备等相关信息。企业能源管控中心通过集中监控作出及时、快速和准确的处置，把能源系统故障所造成的影响控制在最低限度，确保能源系统稳定运行。同时可对一段时间内设备运行时的报警信息进行统计查询，能显示单台设备的详细报警信息，并具备报警确认功能。可作任意时段、任意

工序的报警统计。

# 3) 计划对比

将能源计划与实际消耗量进行对比，掌握企业月度、季度、年度的能源使用状况，可发现能源使用异常状况，做到能源的定量管理。

# 3.4 能源实际管理

对企业实际能源消耗进行月度、季度、年度统计，系统支持按车间、区域等进行能源消耗统计。

# 3.5产值分析

系统可对企业及各车间产值进行分析，新建五车间、后处理车间、溶剂回收车间可与生产模块数据打通，每个月自动计算车间产值消耗，其他车间需企业每个月手动录入产量、产值进行产值消耗计算。

# 3.6 能源分析与优化

通过采集、监控和分析基础能流数据，建立能源预测与优化调度模型，对主要能源介质的生产和消耗进行准确预测，对主要能源介质管网进行在线动态模拟计算，给出重点能源介质优化调度方案，通过能源介质产、存、耗的动态平衡和优化调度，提高能源平衡水平和能源介质利用效率，实现节能减排增效目标。系统主要功能有：

# 1) 分析预测

根据以前的能源消耗数据建立能源预测模型，实时在线了解电、水、蒸汽、天然气等能耗未来变化情况，为能源分配提前做好准备，保证能源充分的利用，使能源充分发挥到最大利用价值，充分分析影响能源利用效率的主要原因，发掘降低能耗的潜为，总结改进措施，明确降低能耗的方向，最终达到能源的高效利用降低污染的目标。

# 2) 重点耗能设备分析

对于 70kw 以上的用电设备需要单独计量。通过用电数据对比（包括同一设备不同时间段的耗能对比，相同类型不同设备同一时间段的耗能对比）比较设备能效，分析设备节能潜力，采取节能改造措施。

# 3.7 能源报表统计

# 1) 实时数据监测

能够实现能耗的分类分项计量、管理、统计功能；支持逐时、逐日、逐月、逐年和任意时段数据的查询、分析；5 年以上能耗数据查询、展示和对比分析；能耗结构、能耗趋势、指标对比展示；为任意对象（企业、车间）任意时段（日月年）的电耗提供饼图、柱形图、曲线图展示、管理和报表汇总、打印功能，并支持 excel 格式的导出等。

![image](https://cdn-mineru.openxlab.org.cn/result/2026-04-13/99fff4eb-6a25-44d8-9563-8eef591fe835/d9ce66c996db7b0c244db633193092b96d8476c67eb280675a5600f07546e55c.jpg)


# 2) 统计报表

系统为用户提供强大的报表功能。主要包括综合能源消耗汇总表（日、周、月、年度报表）、能源生产与消费表、指标汇总统计等。提供自定义报表输出功能，报表支持 EXCEL、PDF 等多种导出格式，用户可以方便的进行编辑和打印。系统提供如下方式生成各种适用报表，实现报表的打印输出功能。

$\diamond$ 能够按照日、周、月、年等形式生成统计图表。

$\diamond$ 能够按照企业、分厂、部门、班组、个人等组织形式对能耗数据进行统计分析。

$\diamond$ 与能耗绩效考核、对标对比分析相结合生成丰富的能耗专家报表系统。

$\diamond$ 具备自定义报表功能，以便管理者自定义生成报表。

# 3.8 系统管理

# 1) 权限管理

具有灵活的权限管理功能，可以根据角色分配业务模块，并能设置每个业务模块数据的增加、删除、修改、导入、导出和打印等操作权限；还可以根据需要对个别用户进行单独的权限设置；不同身份人员可以按照权限设置的范围管理能耗监测设备，查看能耗统计、汇总和分析数据，实现分厂车间、监测设备和能耗数据的分级管理。能从企业和监测设备等多种视角监测管理能耗信息。

# 2) 能源设备

通过对企业能耗设备统一分类，以唯一标识体系为纽带，建立企业生产设备管理整体框架，实现能耗设备台账管理变更管理等，实时掌握设备的状态及设备运行效率，及时淘汰落后设备，避免生产重大事故。
